== Description ==

Title to Titletag uses the title you can give in the media gallery as a title-tag in images.

== Installation ==

1. Upload 'title-to-titletag.zip' through Wordpress 'Plugins' installation menu
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why using Title to Titletag? =

It adds a title tag to your images. These enable a tooltip like hover effect on you images.
This also gives your user more information about an image.

= How does it work? =

Title to Titletag reads the title you give in the image gallery and generates a title-tag from it.
